Multi Clash PokemonJAVA

Este Proyecto se realizó en agosto del 2022 con el fin de entregar un proyecto de la materia Sistemas Distribuidos, lo que se requería era hacer un videojuego en JAVA que posteriormente se pudiera expandir para hacerlo distribuido. En este proyecto lo que puedes hacer es jugar el Sistema de batalla de pokemon pero creando los personajes que tu quieras, modificando el tipo de personajes que hay, con nuevos ataques y un sistema de gestion de pokemon.

¿En Qué Consiste?

MultiClash como se dijo anteriormente es un proyecto sencillo realizado en JAVA, la idea principal fue hacer el sistema de combate Pokemon pero con diversos Personajes al estilo de MUGEN En este Proyecto se utilizaron librerias como lo son Jswing y Math, este proyecto tambien contó con la ayuda del editor Online PIXLR para la edicion de las imagenes así como el diseño general de la interfaz, El IDE utilizado para este Proyecto fue Eclipse IDE for Java Web Developers . Para Correr el Proyecto se necesita tener Java 8 en adelante.
